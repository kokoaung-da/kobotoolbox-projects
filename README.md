# Kobo Toolbox XLSForm Projects

This project shows my experience in creating survey forms in Kobo Toolbox using XLSForm (Excel format).
I made two Excel files that include cascading questions, which means the next question depends on the answer from the previous one.

These forms can be uploaded directly to Kobo Toolbox or ODK to create smart digital surveys.
